15. Propriedades (props) e children de componentes no React

Tudo que vem dentro de um componente React é passado para ele como 'children',
o qual é a primeira propriedade do objeto recebido no primeiro parâmetro do
componente.

export function MyComponent(props) {
  return props.children;
}
