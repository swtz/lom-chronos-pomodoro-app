Nessa aula, há uma técnica muito interessante para alterar componentes com base
no estado atual usando um objeto.

O que motiva essa técnica? O uso da seguinte condicional:

if(state === 'foo') {
  <MyComponent />
} else {
  <AnotherComponent />
}

Com a técnica:

const myObject = {
  foo: <MyComponent />,
  bar: <AnotherComponent />
};

<h1>{myObject[state]}</h1>

Assim, dependendo do valor de 'state', um componente será renderizado.
Dessa maneira, a renderização do componente fica dinâmica, além do código
ficar mais elegante.
