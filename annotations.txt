Aula: 78. RouterLink - Criando o seu próprio Link para o React Router

É possível atribuir a um 'type'o tipo de um componente React:

type MyComponentProps = {
  href: string;
  children: React.ReactNode;
} & React.ComponentProps<'<a>'>;

export function MyComponent({ href, children, ...props }: MyComponentProps) {
  return <Link to={href} {...props}>{children}</Link>;
}

É interessante pegar o 'restante' das 'props', a fim de que o componente personalizado
reconheça todas as 'props' que o componente JSX 'a' possui.
